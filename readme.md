# My First Web Calculator

 This is a simple, interactive calculator web app that I built to practice my frontend development skills. It runs entirely in the browser and handles all the basic math operations you need daily.


## What I Built & How It Works

I built this project using the "Big Three" of web development:
1. **HTML (The Bones):** I structured the calculator layout, setting up an input screen to show the numbers and a grid layout for all the buttons.
2. **CSS (The Looks):** I styled the calculator to make it look clean, modern, and easy on the eyes (and made sure the buttons actually feel satisfying to click!).
3. **JavaScript (The Brains):** This is where the magic happens. I wrote logic to:
   * Grab the number and operator buttons when you click them.
   * Append those characters to the screen dynamically.
   * Run the actual math calculations when you hit `=` and clear the screen when you hit `C`.
